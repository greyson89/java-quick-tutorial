## 型態

- 主要分為兩種
	1. 基本型態 primitive type
	2. 類別型態（參考型態） class type or reference type

- 基本型態
  - 整數
  	1. short	16	-32768 ~ 32767
  	2. int		32	-2,147,483,648 ~ 2,147,483,647
  	3. long		64	-9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807

  - 位元組
	1. byte		8	-128 ~ 127

  - 浮點數
	1. float	32	
	2. double	64

  - 字元
    1. char		16
    2. 使用' '

  - 布林
    1. boolean	

- 類別型態
  - 字串 String
  - 自訂的class
